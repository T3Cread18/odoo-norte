🔌 SmartOLT Integration

🚀 Integración Completa con SmartOLT

Módulo de Odoo que integra completamente con la API de SmartOLT para gestionar equipos GPON de manera eficiente.

📡 Gestión de Equipos GPON

  • OLTs: Administración de equipos principales de la red
  • ONUs: Control completo de equipos de cliente
  • Zonas: Organización geográfica de la red
  • ODBs: Gestión de cajas de distribución óptica

⚙️ Configuración Avanzada

  • Perfiles de Velocidad: Configuración de ancho de banda
  • VLANs: Segmentación del tráfico de red
  • TR069: Gestión remota de equipos
  • VoIP: Configuración de telefonía IP

🔄 Sincronización Automática

  • Sincronización bidireccional con la API de SmartOLT
  • Wizard de sincronización fácil de usar
  • Actualización en tiempo real de estados
  • Manejo automático de errores y reintentos

⚠️ Configuración Requerida

Antes de usar el módulo, debes configurar tu token de API de SmartOLT en los parámetros del sistema.

Clave: smartolt.api_token

🎯 Casos de Uso

  • Proveedores de servicios de internet (ISPs)
  • Empresas de telecomunicaciones
  • Administradores de redes GPON
  • Técnicos de campo y soporte

🔧 Características Técnicas

  • Compatible con Odoo 16.0
  • Interfaz de usuario intuitiva
  • Sistema de permisos granular
  • Logging completo para auditoría
  • Manejo robusto de errores de API
